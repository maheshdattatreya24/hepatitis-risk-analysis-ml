# 🧠 Hidden Pattern Mining in Blood Tests (Unsupervised ML)

## 📌 Overview
This project presents an unsupervised machine learning framework to identify hidden health risk patterns from routine blood test data.

## 🚀 Key Features
- Interaction-based feature engineering
- Isolation Forest for anomaly detection
- PCA for dimensionality reduction
- Explainable AI using perturbation analysis
- Risk score generation (0–1 scale)

## 📊 Dataset
- UCI Hepatitis Dataset
- 155 patient records
- Includes liver function and clinical features

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib

## 📈 Outputs
- Risk score distribution visualization
- Blood test interaction heatmap

## 🎯 Applications
- Early health risk detection
- Preventive healthcare systems
- Clinical decision support
